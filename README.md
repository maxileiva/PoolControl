POOL CONTROL (Android - Jetpack Compose).
Aplicación móvil para reservas y arriendos de piscina/recinto.

Funcionalidades clave (por rol)
Cliente: Agendamiento de arriendos (datos propios de perfil), Consulta arriendos, Funcionalidades para edicion de su perfil
Administrador: Agendamiento de arriendos datos personalizables, consulta y elimancion de arriendos, funcionalidades de edicion de su perfil
Operador: Agendamiento de arriendos datos personalizables, consulta y elimancion de arriendos, funcionalidades de edicion de su perfil

UI: Jetpack Compose + Navigation; pantallas -> Screens. Estados con ViewModel.
Validaciones: email, contraseñas, teléfonos, fechas y bloqueo de calendario, nombres; helpers de navegacion.

Fechas de arriendo: formulario con calendario interactivo (bloquea dias arrendados) (no se pueden arrendar dias pasados).
